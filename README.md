AI EMAIL TRIAGE ASSISSTANT


OVERVIEW-
This project is a simple AI-powered email assistant built using Python and Streamlit.  
It compresses email threads, categorizes messages, auto-generates draft replies, and organizes emails into smart folders.

FEATURES-
- Compress Email Threads: Summarizes long email conversations.
- Categorize Emails: Sorts messages into Work, Personal, Urgent, and Spam.
- Auto-Respond Drafts: Generates suggested replies for quick responses.
- Smart Folder View: Displays emails neatly in folders with key metadata.

TECH STACK-
- Python 3.13
- Streamlit
- Local LLM (Ollama)
- Pandas, NumPy, JSON
